Omnidirectional robot state equations

The goal of this class is to implement the docking of an omnidirectional robot into another robot, based on the state equations of an omnidirectional robot proposed in "Automation for Robotics, Luc Jaulin, 2015."

You are given a copy of this book and a ready to import simulation environment available at:Omnirobot github repository

The environment contains

  1. 3D models of two imaginary robots.
  2. a ready to fill omnidirectional actuator

Tasks

  1. Modify the actuator to implement the omnidirectional robot state equations with the help of [1],[2]
  2. Test the actuator and submit a pull request to http://github.com/nicolaje/omni.
  3. The omni robot should be equiped with a semantic camera, to simulate the result of a complex video processing to detect the second robot.
  4. Using the position of the second robot detected by the semantic camera in the omnidirectional robot frame, control the omni-robot so that it is located 1 meter in front of the second robot. The second robot should always remain in the semantic camera field of view...
  5. Make the second robot move slowly to test your regulator: the omnidirectional robot should remain approximately 1meter away from the second.
